V1.1

Mejoras implementadas

- Se agregó validación para evitar el ingreso de nombres vacíos, asegurando una mejor calidad de los datos.
- Se normalizó el formato del RUT, convirtiéndolo automáticamente a mayúsculas y eliminando espacios innecesarios, lo  que evita duplicados por diferencias de formato.
-Se mejoró la visualización del estudiante encontrado, mostrando la información de manera clara y ordenada para una mejor experiencia de usuario.
